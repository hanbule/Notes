1.1
List containers:
      docker ps -a

1.2
List container IDs:
      docker ps -aq

1.3
Remove containes:
      docker rm <container ID>

2.1
List images:
	docker images

2.2
List images:
	docker images -q

2.3
Remove image:
	docker rmi <image ID>


4
To check docker info:
	sudo docker info

5.1
Check if docker is active: 
	systemctl is-active docker

5.2
Check docker status:
	systemctl status docker
	or
	service status docker
	
6
Start/stop docker:
	systemctl start/stop docker
	or
	service start/stop docker
