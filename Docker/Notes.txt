1.1
List containers:
      docker ps -a

1.2
List container IDs:
      docker ps -aq

1.3
Remove containes:
      docker rm <container ID>

2.1
List images:
	docker images

2.2
List images:
	docker images -q

2.3
Remove image:
	docker rmi <image ID>

3
Start/stop docker (ubuntu):
	systemctl start/stop docker
	or
	service start/stop docker

4
Start docker image:
	docker run <options> <image name>

5.1
To check docker info:
	sudo docker info

5.2
Check if docker is active (ubuntu): 
	systemctl is-active docker

5.3
Check docker status (ubuntu):
	systemctl status docker
	or
	service status docker
	
6
Access docker container:
	a - start container of interest first
	b - look up <container id> by command 1.1 above
	b - run in terminal:
	    docker exec -it <container id> bash
